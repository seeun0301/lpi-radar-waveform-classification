# Lightweight CNN for LPI Radar Waveform Classification

This repository provides a Python implementation of a lightweight convolutional neural network (CNN) for Low Probability of Intercept (LPI) radar waveform classification.

## Overview

Low Probability of Intercept (LPI) radar waveforms are widely used in modern radar systems due to their anti-detection characteristics.

This project investigates lightweight CNN architectures for efficient LPI radar waveform identification while maintaining high classification performance.

## Features

- LPI radar waveform preprocessing
- Lightweight CNN architecture
- Training and evaluation pipeline
- Classification performance analysis
- Reproducible experiments
